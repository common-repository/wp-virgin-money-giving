A widget to show your Virgin Money Giving information.

== Description ==

There are two things you'll need to start using the WordPress Virgin Money Giving plug-in, a fundraisers API key and a fundraisers API reference.
Once in place add the "Virgin Giving Fundraisers Page" widget to anyone of your sidebars, the widget will have options for you to select in order for the widget to function correctly.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `wp-virgin-money-giving` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. For the `fundraisers API key` sign in or register a developers account at the <a href="https://developer.virginmoneygiving.com/">developers virgin money giving website</a>. With your developers account created use the `get a fundraisers API key` link to create your fundraisers API key. All your API keys will be listed under the "My Account" link.
4. For the `fundraisers API reference` simply login to your personal Virgin Money Giving account and you'll find your `fundraisers API reference` in your main account page, it's referenced as `My fundraisers API reference:`


== Frequently Asked Questions ==

= I don't have a Virgin Developers Account =

Simply go to https://developer.virginmoneygiving.com/ and register your developers account.

= Where can I find my 'fundraisers API key' and 'fundraisers API reference' =

The API key can be found in your Developers Virgin Money Giving account, all your API keys will be listed under the "My Account" link.
The API reference can be found in your personal Virgin Money Giving account main page, it's referenced as "My fundraisers API reference:"
